The BlackMagic framework supports the following configuration options:

  • themeClass - CSS class to toggle instead of dynamic color conversion
  • factor - Intensity of color changes (0.1 = subtle, 0.8 = dramatic)
  • backgroundColor - Custom dark mode background color
  • autoSwitch - Automatically apply saved theme on page load
  • cookieDuration - How long to remember user's theme preference
  • cookieName - Custom cookie name for theme storage
